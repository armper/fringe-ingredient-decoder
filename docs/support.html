Common fixes

  • The app opens to a home screen first. Tap Scan barcode when you want to use the camera.
  • If barcode lookup fails, paste the ingredient text instead.
  • If a label looks incomplete, try copying the full ingredients section including items inside parentheses.
  • Some unfamiliar ingredients may sharpen a moment after the first result appears because the app quietly checks a public ingredient reference and stores the better match locally.
  • If a recent item is outdated, rescan or paste the current label again.

Data sources

Ingredient matches are powered by public reference data from Open Food Facts, Open Beauty Facts, and the FDA inactive ingredients database, combined with on-device parsing, local caching, and on-device scoring.
